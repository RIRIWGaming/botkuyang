## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> pkg update && pkg upgrade 
> pkg install nodejs
> pkg install tesseract 
> pkg install git
> pkg install ffmpeg
> git clone https://github.com/RIRIWGaming/botkuyang
> cd botkuyang
> bash install.sh
```

> node ririwgaming.js
